# 行为准则

## 我们的承诺

作为社区的一员、贡献者和领导者，我们致力于让每个人都能在我们的社区中获得友好和包容的体验。

## 我们的标准

积极的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验

不可接受的行为包括：

- 使用贬损性或侮辱性的评论
- 公开或私下的骚扰

## 我们的责任

项目维护者有责任澄清可接受的行为标准，并对任何不可接受的行为采取适当和公正的纠正措施。

## 适用范围

本行为准则适用于所有社区场所。

## 执行

如有辱骂、骚扰或其他不可接受的行为，可通过联系项目团队来报告。
